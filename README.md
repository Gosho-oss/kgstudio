# KG Studio

This repository contains the source code for the KG Studio portfolio website and its client projects.

## About

KG Studio is a web design and development studio that specializes in creating modern, high-converting websites for small businesses.

## Projects

Here are some of the projects included in this portfolio:

-   **[Urban Fork Restaurant](urban-fork.html)**: A modern booking platform for a local bistro.
-   **[IronPulse Fitness](ironpulse.html)**: Membership portal and class scheduler for a gym.
-   **[Apex Financial](apex-financial.html)**: Corporate identity and lead gen for a firm.
-   **[DriveCare Auto](drivecare.html)**: Service menu and appointment booking integration.

## How to View

1.  Clone this repository.
2.  Open the `index.html` file in your web browser to view the main portfolio page.
3.  You can also open each project's HTML file directly to view it.
